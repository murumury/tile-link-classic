# Privacy Policy

Tile Link Classic is a pure local Chrome extension game.

## Data Collection

The extension does not collect, transmit, sell, share, or rent user data.

## Local Storage

The extension uses `chrome.storage.local` only to save game settings, language preference, difficulty preference, visual options, and local best scores on the user's own device.

This data stays in the user's browser profile and is not sent to any server.

## Permissions

The extension requests the `storage` permission so it can preserve local game settings and best scores between sessions.

It does not request host permissions, does not read or modify websites, and does not inject content scripts by default.

## Network Access

The game is bundled with local assets and does not make network requests during gameplay.

## Contact

For privacy questions, use the support contact listed on the Chrome Web Store listing or the project repository.
